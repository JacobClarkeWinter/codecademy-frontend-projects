HTML Table Cheat Sheet

Table tags

Tag Name Description
<table> Table Wrapper element for HTML tables
<tr> Table Row defines a row of cells in a table. The row's cells can then be established using a mix of data cell and header cell elements
<th> Table Header Cell Defines a cell as the header of a group of table cells. The exact nature of this group is defined by the scope and headers attributes
<td> Table Data Cell Defines a cell of a table that contains data. It participates in the table model
<thead> Table Head Defines a set of rows defining the head of the columns of the table
<tbody> Table Body Encapsulates a set of table rows, indicating that they comprise the body of the table
<tfoot> Table Footer Defines a set of rows summarizing the columns of the table

Table Attributes

Attribute Name Description
colspan Column Span Makes a cell span over multiple columns
rowspan Row Span Makes a cell span over multiple rows
